测试覆盖率

需要安装python-dev 开发包，否则安装会失败

pip install coverage

nose example 测试生成测试数据 add.py 


def add(a, b):
    return a + b

test_add.py 


import add
def test_add():
    assert add.add(3,4) == 7

 nosetests --with-coverage --cover-erase --cover-package=add
coverage 生成html 可视化测试报告 
coverage html
